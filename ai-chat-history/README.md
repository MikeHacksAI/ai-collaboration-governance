# AI Chat History Logging

## Purpose
This folder stores transcripts of AI chat sessions for audit, review, and collaboration.

## How to Save Chat History
1. After each session, copy the chat transcript.
2. Create a new markdown file:
	ai-chat-history/session-YYYY-MM-DD.md
	(Replace YYYY-MM-DD with the session date.)
3. Paste the transcript into the file.
4. Optionally, add a summary or tags at the top.

## Best Practices
- Use clear filenames with dates for easy tracking.
- Add a brief session summary at the top of each file.
- Link relevant chat logs to incident or drift records as needed.

## Template
`
# AI Chat Session – YYYY-MM-DD

**Summary:**
[Brief description of session purpose/outcome]

---

[Paste chat transcript here]
`
