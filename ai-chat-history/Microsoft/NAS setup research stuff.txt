Recommendation for Your Current Setup

Since you have two 12TB drives and want to maximize space, you should still consider 
using SHR for both separate pools instead of Basic.
Even though you can't add a third drive, SHR offers more flexibility if you ever decide to 
move these drives to a larger 4-bay Synology NAS in the future. In a larger unit, an SHR pool 
can be expanded by simply sliding in more drives, whereas a "Basic" pool would have to be 
converted manually. 
Synology Community

Set up your storage pools first. You can't configure rclone or backups until the NAS 
actually has a "place" to store data and run applications.
Since you want two separate 12TB drives, follow these steps in Storage Manager to ensure 
Synology doesn't accidentally "help" you by mirroring them into a single 12TB pool.
Step 1: Create the First Pool (Drive 1)
1.	Open Storage Manager from the DSM main menu.
2.	Go to the Storage tab and click Create > Create Storage Pool.
3.	RAID Type: Select SHR (Synology Hybrid RAID).
4.	Select Drive: Check only Drive 1.
5.	Follow the wizard to completion. It will prompt you to create a Volume immediately 
after—choose the Btrfs file system.
Step 2: Create the Second Pool (Drive 2)
1.	Click Create > Create Storage Pool again.
2.	RAID Type: Select SHR again.
3.	Select Drive: This time, check only Drive 2.
4.	Complete the wizard and create a second Volume (also Btrfs).
Why this order matters
*	Logical Separation: By creating two pools, you guarantee that if Drive 1 dies, Drive 
2 stays online.
*	Btrfs Check: Selecting Btrfs via Synology is critical for your DS223 because it allows 
for Snapshots, which protect your data from accidental deletion or 
ransomware before it even gets synced to OpenDrive.

To set up your Synology DS223 with two separate 12TB drives, you will need to create two 
independent storage pools.
Recommended RAID Choice
Choose SHR (Synology Hybrid RAID) for both drives, but set them up as separate pools. 
While you can choose "Basic," SHR is the better option for these reasons:
Identical Performance: A single-drive SHR pool performs exactly the same as a "Basic" 
pool.
Portability: If you ever upgrade to a larger 4-bay or 5-bay Synology NAS in the future, SHR 
allows you to simply add more drives to that pool later to gain data protection without 
reformatting.

Step-by-Step Setup
Do not follow the "Quick" setup, as it will likely try to mirror your two drives into a single 
12TB pool (RAID 1). Instead, follow these steps in the Synology Storage Manager: 
Create Pool 1:
Select Create > Create Storage Pool.
Choose SHR as the RAID type.
Select only one of your 12TB drives.
Complete the wizard and create a Volume on this pool.
Create Pool 2:
Repeat the process: Create > Create Storage Pool.
Choose SHR again.
Select the second 12TB drive.
Create a second Volume on this pool. 

Important Considerations
No Redundancy: Since each drive is its own pool, you will have the full 24TB total space, 
but if one drive fails, you will lose all data on that specific drive.
Btrfs File System: When creating your volumes, choose Btrfs (instead of ext4) to enable 
features like snapshots and file self-healing.
Shared Folders: When you create a Shared Folder, you will now have a dropdown menu to 
choose whether it lives on Volume 1 (Drive 1) or Volume 2 (Drive 2).
