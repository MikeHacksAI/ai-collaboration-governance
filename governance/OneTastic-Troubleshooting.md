✅ Fix 1 — Turn OFF “Simplified Ribbon”
This is the #1 cause.
In OneNote:
- Look at the top-right of the ribbon
- If you see a toggle called Simplified Ribbon, turn it off
The Onetastic tab only appears in the full ribbon.

✅ Fix 2 — Reset the ribbon customizations
- File → Options
- Customize Ribbon
- Bottom-right: Reset → Reset all customizations
- Restart OneNote
This forces the Onetastic tab to re-register.

✅ Fix 3 — Check if the Onetastic tab is unchecked
Same screen:
- File → Options → Customize Ribbon
- On the right side, scroll the list
- Look for Onetastic or Macros
- Make sure the checkbox is ON
- Restart OneNote

✅ Fix 4 — Force OneNote to reload the add‑in UI
Even though the COM add‑in is active, the UI sometimes doesn’t load.
Do this:
- Close OneNote completely
- Open Task Manager → kill OneNote.exe
- Reopen OneNote Desktop
- The Onetastic tab should appear
