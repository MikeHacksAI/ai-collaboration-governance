# SharePoint Online Modern Intranet Lab Case Study

## Status

In progress. This is a personal validation lab, not client production work.

## Public-Safe Summary

Built and documented a SharePoint Online lab to validate modern intranet implementation workflow. The lab focuses on site structure, library metadata, permission model, and migration validation. Evidence includes configuration screenshots, metadata views, and post-build validation notes.

## Objective

Demonstrate practical SharePoint Online implementation readiness using a controlled Microsoft 365 developer tenant.

## Environment

- Tenant: Microsoft 365 Developer Program (E5 sandbox)
- Platform: SharePoint Online modern experience
- Source data: Local Windows folder
- Tools: SharePoint admin center, SharePoint Migration Tool (SPMT), optional PnP PowerShell
- Cost: $0

## Scenario

- Organization: Northwind Ops (simulated)
- Target site: Operations Hub
- Core libraries:
  - Policies
  - Projects
  - Vendor Documents
- Metadata model:
  - Department
  - Document Type
  - Fiscal Year
  - Status
